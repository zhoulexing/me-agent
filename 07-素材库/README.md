# 素材库

这个目录用于积累和整理各类有价值的素材，包括但不限于：
- 公众号文章
- 视频内容
- 博客文章
- 技术文档
- 其他有价值的网络内容

## 文件命名规范

文件名格式：`YYYY-MM-DD.md`

示例：`2026-03-24.md`

同一天的所有素材积累在同一个文件中。

## 素材格式

每个素材包含以下部分：

```markdown
## 标题

素材的标题或主题

### 总结

对素材内容的概括、关键观点、个人思考等

### 原始素材

- **来源**：公众号名称 / 网站名称 / 视频平台等
- **链接**：原始链接地址
- **日期**：收集日期（可选）
```

## 示例

```markdown
## AI时代计算机专业学习的价值与思维培养

### 总结

在AI时代，计算机专业的价值不再局限于实操技能，更重要的是培养计算思维。卡斯克提到，设计课程分为实操技能课和基础理论课。实操技能容易被AI替代，而基础理论（如审美、品味、判断力）在AI时代反而更加重要。

同样，计算机专业的核心价值在于：
- 把复杂问题拆解的能力
- 找到逻辑和规律的能力
- 用模型推演结果的能力
- 判断力和审美能力

这些是AI无法替代的核心竞争力。

### 原始素材

- **来源**：数字生命卡斯克 × 快刀青衣 对谈
- **链接**：[视频链接地址]
- **日期**：2026-03-23
```

## 使用建议

1. **及时收集**：遇到有价值的素材及时记录，避免遗漏
2. **深度思考**：在总结部分加入自己的理解和思考
3. **定期回顾**：定期翻看素材库，加深理解和记忆
4. **分类标记**：可以使用标签或目录结构对不同类型素材进行分类
