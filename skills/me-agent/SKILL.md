## 工作流程

### 1. 日记管理

**触发词识别**："记日记"、"添加到流水账"、"今天发生了什么"、"记录一下今天"

**操作步骤**：
1. 确认日期（使用当前日期 YYYY-MM-DD）
2. 检查 `~/clawd/project/me-agent/06-流水账/YYYY-MM-DD.md` 是否存在
3. 文件不存在 → 创建新文件，写入日记模板
4. 文件已存在 → 追加内容到对应章节

**日记模板**：
```markdown
## 今日事件

## 今日书账

## 今日美食

## 今日新知

## 今日碎记
```

### 2. 信息记录

**触发词识别**："记一下信息"、"保存这个"、"帮我记录"、"记录到信息记录"

**操作步骤**：
1. 确定分类：
   - `experience`（体验类）：餐厅、景点、产品体验等
   - `doc`（文档类）：教程、操作指南、技术文档等
   - `other`（其他）
2. 生成文件名：`YYYY-MM-DD-[分类]-[名称].md`
3. 创建文件到 `~/clawd/project/me-agent/05-信息记录/`
4. 如不确定分类，询问用户

**模板参考**：详见 `references/templates/info-record.md`

### 3. 素材库管理

**触发词识别**："保存素材"、"添加到素材库"、"记录这个文章/视频"

**操作步骤**：
1. 确认日期（YYYY-MM-DD）
2. 检查 `~/clawd/project/me-agent/07-素材库/YYYY-MM-DD.md` 是否存在
3. 追加素材内容：

```markdown
## 标题

### 总结

### 原始素材

- **来源**：公众号/网站/视频平台
- **链接**：原始链接
- **日期**：YYYY-MM-DD
```

4. 同一天的素材积累在同一文件中

### 4. Get 笔记同步

**触发词识别**："同步到 get笔记"、"同步到 Get 笔记"、"添加到知识库"

**操作步骤**：
1. 确认要同步的文件路径
2. 根据路径确定知识库：
   - `02-个人认知/xxx.md` → 知识库 `个人认知`
   - `03-技术认知/xxx.md` → 知识库 `技术认知`
   - `05-信息记录/xxx.md` → 知识库 `信息记录`
3. 调用 Get 笔记技能：
   - 标题：文件名（不含扩展名）
   - 内容：文件正文
   - 知识库：映射后的知识库名称
4. 反馈同步结果

**映射规则**：详见 `references/paths.md`

### 5. 认知查询

**被动触发（基于对话内容）**：

| 触发场景 | 查询目标 | 目录路径 |
|---------|---------|---------|
| 用户说"我觉得"、"我认为" | 个人认知 | `02-个人认知/` |
| 讨论技术方案、架构决策 | 技术认知 | `03-技术认知/` |
| 评估"我能做到吗" | 个人技能 | `04-个人技能/` |
| 查询基本信息 | 基本信息查询 | `01-基本信息/` |

**操作步骤**：
1. 根据对话内容判断触发场景
2. 使用 Glob 工具列出目标目录文件
3. 根据文件名选择相关文档
4. 使用 Read 工具读取文档
5. 基于认知内容给出响应
