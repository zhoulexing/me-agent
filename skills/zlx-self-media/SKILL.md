# Self Media

## Overview
这个 skill 用于处理自媒体相关任务。

当前能力聚焦在“立人设”这个核心场景，帮助用户逐步想清楚自己是谁、适合输出什么、应该以什么样的方式被用户认知。

整体执行上，不追求一次性灌输完整方法论，而是优先通过提问、拆解、澄清与引导，帮助用户自己把关键判断说清楚、想通透。

后续可以继续在这个 skill 里动态补充新的自媒体能力模块，例如选题、内容规划、脚本创作、内容优化、平台适配与复盘。

## References
| Reference | Trigger | Notes |

## Workflows
- 明确用户意图
- 根据用户意图选择对应 reference
- 按照对应的 Reference 要求来回答

## Constraints
- 不要 AI 味太重
- 以苏格拉底式引导为主，循循善诱
- 核心目标不是替用户仓促下结论，而是让用户自己逐步想明白、说清楚、建立通透感
