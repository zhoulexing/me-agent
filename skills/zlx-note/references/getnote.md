# Get 笔记同步规则

## 核心原则

**本地优先，单向同步**：
- **同步到 Get 笔记**：本地有，知识库没有 → 上传
- **从 Get 笔记下载**：知识库有，本地没有 → 下载

**层级对应关系**：
1. **第一层**：知识库与文件夹一一对应
2. **第二层**：在对应范围内比较文件

## 目录映射

| 本地目录 | Get 笔记知识库 |
|---------|---------------|
| `01-基本信息/` | `基本信息` |
| `02-个人认知/` | `个人认知` |
| `03-技术认知/` | `技术认知` |
| `04-素材库/` | `素材库` |
| `05-信息记录/` | `信息记录` |
| `06-流水账/` | `GetSeek录音卡` |

**映射规则**：去掉序号前缀（如 `02-`），保留后半部分作为知识库名

## 同步到 Get 笔记

### 同步条件
- 本地文件存在
- Get 笔记知识库中不存在同名笔记

### 文件名映射

**个人认知/技术认知/信息记录文件**：
- 本地格式：`[主题]-[tag1,tag2]-[YYYY-MM-DD].md`
- 笔记标题：只取 `[主题]` 部分（去除标签和日期）

**示例**：
```
本地文件：AI时代的学习-写作,思考-2026-03-25.md
笔记标题：AI时代的学习
笔记内容：文件完整内容
```

**其他目录文件**（基本信息、素材库、信息记录）：
- 本地格式：直接使用文件名（不含扩展名）
- 笔记标题：文件名

**流水账目录文件**：
- 本地格式：`YYYY-MM-DD.md`
- 对应 Get 端 `GetSeek录音卡` 中同日期主题内容

## 从 Get 笔记下载

### 下载条件
- Get 笔记知识库中存在
- 本地目录中不存在同名文件

### 文件名映射

**个人认知/技术认知/信息记录**：
- 笔记标题 → 本地文件名：`[标题]-[YYYY-MM-DD].md`
- 日期为下载日期

**示例**：
```
笔记标题：AI时代的学习
本地文件：AI时代的学习-2026-03-25.md
```

**其他目录**（基本信息、素材库、信息记录）：
- 笔记标题 → 本地文件名：`[标题].md`

**流水账目录**（`06-流水账/`）：
- `GetSeek录音卡` 中的日期主题内容 → 本地文件名：`[YYYY-MM-DD].md`

## 操作流程

### 同步到 Get 笔记
1. **选择本地文件夹**（如 `02-个人认知/`）
2. **映射到知识库**（如 `个人认知`）
3. **在对应范围内比较**：
   - 获取本地文件夹的文件列表
   - 获取该知识库的笔记列表
   - 找出本地有、知识库没有的文件
4. **提取笔记标题**（去除标签和日期）
5. **调用同步接口上传**

### 从 Get 笔记下载
1. **选择知识库/卡片容器**（如 `个人认知` 或 `GetSeek录音卡`）
2. **映射到本地文件夹**（如 `02-个人认知/` 或 `06-流水账/`）
3. **在对应范围内比较**：
   - 获取该知识库的笔记列表
   - 获取本地文件夹的文件列表
   - 找出知识库有、本地没有的笔记
4. **根据笔记标题生成本地文件名**
5. **调用下载接口保存到本地**
