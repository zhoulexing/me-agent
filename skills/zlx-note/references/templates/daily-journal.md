# 日记模板

**使用场景**：记录每日流水账

**目标目录**：`~/clawd/project/me-agent/06-流水账/`

**文件命名**：`YYYY-MM-DD.md`

---

## 模板内容

```markdown
# YYYY-MM-DD

## 今日事件

时间点：hh:mm
标签：xx
原始内容：xx

----

## 今日书账

时间点：hh:mm
标签：xx
原始内容：xx

----

## 今日美食

时间点：hh:mm
标签：xx
原始内容：xx

----

## 今日新知

时间点：hh:mm
标签：xx
原始内容：xx

----

## 今日素材

记录素材为 xxx

----

## 今日碎记

时间点：hh:mm
标签：xx
原始内容：xx

----

```

---

## 使用说明

### 今日事件
记录当天发生的重要事件、里程碑、转折点等，按“时间点 / 标签 / 原始内容”格式记录

### 今日书账
记录阅读的书籍、文章，包括书名、作者、读后感等，按“时间点 / 标签 / 原始内容”格式记录

### 今日美食
记录品尝的美食，包括餐厅、菜品、个人评价等，按“时间点 / 标签 / 原始内容”格式记录

### 今日新知
记录当天学到的新知识、新技能、新观点等，按“时间点 / 标签 / 原始内容”格式记录

### 今日素材
记录当天收集到的文章、视频、播客、网页等素材，但这里只写简要概要，格式统一为 `记录素材为 xxx`

素材的完整内容需要另外追加到 `~/clawd/project/me-agent/06-流水账/素材.md`，字段固定为：
- `时间：YYYY-MM-DD hh:mm`
- `标题：xxx`
- `摘要：xxx`
- `源链接：xxx`

### 今日碎记
记录其他零散的感想、随笔、待办事项等，按“时间点 / 标签 / 原始内容”格式记录

### 条目分隔
同一个章节里，如果有多条记录，不同记录之间用 `----` 分隔

### 时间点
默认填写当前这条记录实际写入时的时间，例如 `08:50`

---
