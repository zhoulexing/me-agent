## 工作流程

### 1. 基本信息管理

#### 查询基本信息
**触发词识别**："查询基本信息"

**操作步骤**：
1. 使用 Glob 工具列出 `$PROJECT_ROOT/01-基本信息/` 目录文件
2. 根据查询主题选择相关文档
3. 读取文档内容
4. 返回基本信息

#### 记录基本信息
**触发词识别**："记录基本信息"、"更新基本信息"

**操作步骤**：
1. 确认信息分类（如个人资料、联系方式等）
2. 生成文件名：`[信息分类].md`
3. 创建或更新文件到 `$PROJECT_ROOT/01-基本信息/`
4. 使用标准格式记录信息

### 2. 个人认知管理

#### 查询个人认知
**触发词识别**："查询个人认知"

**操作步骤**：
1. 使用 Glob 工具列出 `$PROJECT_ROOT/02-个人认知/` 目录文件
2. 根据对话主题选择相关文档
3. 读取文档内容
4. 基于个人认知给出响应

#### 添加个人认知
**触发词识别**："添加个人认知"、"更新个人认知"

**操作步骤**：
1. 确认认知主题或分类
2. 生成文件名：`[主题名称]-[tag1,tag2]-[YYYY-MM-DD].md`
3. 创建文件到 `$PROJECT_ROOT/02-个人认知/`
4. 将原始内容全部添加到文件，不做任何修改

### 3. 技术认知管理

#### 查询技术认知
**触发词识别**："查询技术认知"

**操作步骤**：
1. 使用 Glob 工具列出 `$PROJECT_ROOT/03-技术认知/` 目录文件
2. 根据技术主题选择相关文档
3. 读取文档内容
4. 基于技术认知给出建议

#### 添加技术认知
**触发词识别**："添加技术认知"、"更新技术认知"

**操作步骤**：
1. 确认技术主题或分类
2. 生成文件名：`[技术主题]-[tag1,tag2]-[YYYY-MM-DD].md`
3. 创建文件到 `$PROJECT_ROOT/03-技术认知/`
4. 将原始内容全部添加到文件，不做任何修改

### 4. 信息记录

#### 查看信息
**触发词识别**："查看信息记录"

**操作步骤**：
1. 使用 Glob 工具列出 `$PROJECT_ROOT/05-信息记录/` 目录下的文件
2. 根据文件名和日期展示信息列表
3. 如需查看具体内容，读取对应文件

#### 记录信息
**触发词识别**："记录信息"、"更新信息"

**操作步骤**：
1. 确定分类：
   - `experience`（体验类）：餐厅、景点、产品体验等
   - `doc`（文档类）：教程、操作指南、技术文档等
   - `other`（其他）
2. 生成文件名：`YYYY-MM-DD-[分类]-[名称].md`
3. 创建文件到 `$PROJECT_ROOT/05-信息记录/`
4. 如不确定分类，询问用户

**模板参考**：详见 `references/templates/info-record.md`

### 5. 流水账管理

#### 查看流水账
**触发词识别**："查看流水账"、"查看素材"

**操作步骤**：
1. 确认日期（默认当前日期，或指定日期）
2. 读取 `$PROJECT_ROOT/06-流水账/YYYY-MM-DD.md`
3. 用户如果查看素材，优先展示 `## 今日素材` 章节；没有该章节时明确说明
4. 用户如果查看流水账，展示完整内容

#### 添加流水账
**触发词识别**："添加到流水账"、"保存素材"、"添加素材"、"更新素材"

**操作步骤**：
1. 确认日期（使用当前日期 YYYY-MM-DD）
2. 检查 `$PROJECT_ROOT/06-流水账/YYYY-MM-DD.md` 是否存在
3. 文件不存在 → 参考模版创建新文件
4. 文件已存在 → 追加内容到对应章节
5. 如果用户记录的是素材，统一追加到 `## 今日素材` 章节，保留素材的标题、总结、来源、链接、日期等结构

**模板参考**：详见 `references/templates/daily-journal.md`

### 6. 笔记同步

#### 同步到Get笔记
**触发词识别**："同步到 get笔记"

**操作步骤**：
1. 确认要同步的文件路径
2. 根据路径确定知识库：
   - `02-个人认知/xxx.md` → 知识库 `个人认知`
   - `03-技术认知/xxx.md` → 知识库 `技术认知`
   - `05-信息记录/xxx.md` → 知识库 `信息记录`
   - 其他同上 
3. 调用 `getnote` 技能：
   - 标题：文件名（不含扩展名如tag和时间）
   - 内容：文件正文
   - 知识库：映射后的知识库名称
4. 反馈同步结果

**映射规则**：详见 `references/paths.md`

#### Get笔记下载
**触发词识别**："Get笔记下载"

**操作步骤**：
1. 确认要下载的知识库和笔记
2. 调用 Get 笔记技能下载笔记内容
3. 根据内容确定本地路径：
   - 知识库 `个人认知` → `02-个人认知/`
   - 知识库 `技术认知` → `03-技术认知/`
   - 知识库 `信息记录` → `05-信息记录/`
4. 创建或更新本地文件
5. 反馈下载结果
