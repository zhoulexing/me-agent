# 标题原则

## 目标

生成让人想点击、想阅读、想观看的标题。好标题不是平铺直叙地概括内容，而是把内容包装成一个共性问题、具体场景、真实冲突，或者一个新鲜视角。

## 核心规则

### 1. 先写用户想看什么，不要只写作者想表达什么

较弱：
- 如何做西红柿炒蛋
- 坚持发视频30天希望成为更好的自己
- 做自媒体好难啊，又没流量了

较强：
- 为什么饭店的西红柿炒鸡蛋更香
- 坚持日更30天，我意外收获了比流量更重要的东西
- 做自媒体三个月，我终于发现了没流量的真正原因

### 2. 优先抛出共性问题，不要一上来就把答案说完

较弱：
- 孩子沉迷手机要怎么管

较强：
- 为什么你越不让孩子玩手机，他越想方设法地偷着玩？

### 3. 优先讲故事和场景，不要空讲道理

较弱：
- 远亲不如近邻

较强：
- 因为一袋垃圾，我跟邻居10年没说过话，直到昨天发生了……

较弱：
- 当妈以后要控制情绪真的很难

较强：
- 今天我在超市，对着3岁的儿子崩溃大哭，因为……

### 4. 优先给出使用场景，不要泛泛而谈

较弱：
- 如何处理人际关系

较强：
- 用这个方法，我让菜市场最难说话的老板主动给我抹了零头

### 5. 保留真实悬念，不要写成标题党

好的悬念必须建立在真实内容支撑上，而不是靠空泛夸张。

例如：
- 做短视频的5点收获，尤其是第5点

它之所以有效，是因为：
- 有数字，结构清楚
- 有结果，价值明确
- 有悬念，但没有把内容一次说透

## 质量检查清单

一个较强的标题，通常至少具备下面 2 到 3 项：
- 用户视角
- 具体场景
- 反差或信息差
- 情绪张力

## 标题类型

生成时尽量混合使用这些类型：

### 惊叹式
- 强调惊讶感或强反差

### 数字型
- 用数字提升清晰度和结构感

### 价格锚定
- 在适合的场景下使用价格与价值反差

### 悬念式
- 制造好奇心，但不要提前剧透完整答案

### 时机场景
- 把标题落在时间点或具体使用场景上

### 颠覆认知
- 给出反直觉的判断或新认知

### 争议式
- 通过强对比或强结果制造讨论张力

### 共鸣式
- 命中受众共同的困扰、焦虑或欲望

## 输出格式

按下面的格式输出：

```text
【惊叹式】标题A
【数字型】标题B
【悬念式】标题C
```

## 最后提醒

不要只写概括内容的标题。
要写出“别人为什么会想点开”的理由。
