# 每日流水账

记录每一天的生活点滴，建立个人生活的数字化档案。

## 📝 日记模板

### 今日事件
见何人、去何地、做何事

**示例：**
- 10点去咖啡馆见XX，聊合作方案
- 下午3点参加产品评审会议
- 晚上7点和家人视频通话

### 今日书账
读/买/推荐的书，含页码与要点

**示例：**
- 读《XX》p50-80，讲XX理论
- 买了《创新者的窘境》
- 推荐朋友看《深度工作》

### 今日美食
探店/外卖，记店名、菜品、口味

**示例：**
- XX餐厅，番茄牛腩，偏甜
- 楼下便利店，关东煮，萝卜很好吃
- 自己做，清炒时蔬，火候刚好

### 今日新知
冷知识、专业常识、八卦

**示例：**
- 蜂蜜永远不会变质，考古发现千年蜂蜜仍可食用
- Python 3.12 引入了新的类型系统改进
- 听说XX公司要被收购了

### 今日碎记
想表达的杂话，情绪与思考

**示例：**
- 今天状态不错，工作效率高
- 对某个问题的思考
- 记录一个小灵感

## 📂 文件命名规范

建议使用日期命名：`YYYY-MM-DD.md`

**示例：**
- `2026-03-22.md`
- `2026-03-21.md`

## 💡 使用建议

1. **保持简洁**：不必每项都填，有内容的就写
2. **及时记录**：当天记录，避免遗忘
3. **真实自然**：流水账是给自己看的，真实最重要
4. **定期回顾**：每周或每月翻看，回顾生活轨迹

## 📊 数据价值

- 追踪时间分配
- 发现生活习惯
- 记录成长轨迹
- 保存美好回忆
