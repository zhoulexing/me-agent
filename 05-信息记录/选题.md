# OpenClaw 飞书插件 — 自媒体文字稿选题

## 项目背景

**OpenClaw Lark Plugin** 是字节跳动 Lark 团队开发的飞书 AI Agent 插件，将 OpenClaw AI 无缝对接到飞书工作区，赋予 AI 直接读写飞书各类应用的能力。npm 包名：`@larksuite/openclaw-lark`，技术栈：TypeScript + Node.js v22+ + 飞书官方 SDK。

# OpenClaw 主体项目 — 自媒体文字稿选题

## 项目背景

**OpenClaw** 是一个运行在自有设备上的**个人 AI 网关**，通过用户已有的消息渠道（WhatsApp、Telegram、Discord、Slack、飞书、微信等 30+ 渠道）进行交互，支持语音通话和实时 Canvas 渲染。npm 包名：`openclaw`，技术栈：TypeScript + Node.js v22+。

定位：本地化、始终在线的个人 AI 助理，所有数据留在本地，支持插件扩展。
