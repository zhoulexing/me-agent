# 技术能力

本目录包含技术能力相关的文档，分为实操类和原理类。

## 📂 子目录

### 实操类

- `架构设计/` - 架构设计实践（后端、前端）
- `智能体设计/` - Agent设计实践（架构、提示词、脚手架）
- `工程实践/` - 开发流程、最佳实践

### 原理类

- `架构理解/` - 对不同架构的理解、对比
- `AI底层原理/` - 深度学习、大模型、多模态
- `技术选型/` - 框架、工具对比

## 🔄 版本管理

- **更新频率**: 季度或技术重大变化时
- **版本策略**: 重大技术变化时升级主版本
